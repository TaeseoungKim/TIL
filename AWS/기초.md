## Chapter 01. 서버리스 정적 웹사이트 호스팅 및 성능 가속화

1. Chapter 개요
   서버가 없어도 구성이 가능한 정적 웹 사이트를 호스팅하고, 콘텐츠 전송 네트워크(CDN) 서비스를 이용하여 웹 사이트의 성능을 향상시킵니다.
2. 사용하는 AWS 서비스

   - Amazon S3 (Simple Storage Service)
   - Amazon CloudFront(콘텐츠 전송 서비스: CDN)
